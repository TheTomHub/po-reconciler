Why Chandlr

Built for order processing teams

Works with any ERP

SAP, Oracle, D365, NetSuite. Chandlr stages data in a clean sheet — you enter it however you normally do. No API connectors required.

Prefix SKU matching

PO says "1046", ERP says "1046V001"? Matched automatically. Case-insensitive, whitespace-tolerant.

SharePoint price lists

The Teams agent automatically fetches the latest price list from your team’s SharePoint. Always up to date. No manual downloads.

Full audit trail

Every output shows which ERP record matched each line. Results, credit notes, re-invoices, ERP staging — all preserved for review.

Data quality gate

9 validation rules catch duplicates, missing prices, quantity outliers, and line total mismatches before reconciliation runs.

ACCEPT / REVIEW / ESCALATE

After every reconciliation, Chandlr gives a risk recommendation based on exception rate, exposure, and historical patterns for that customer.
