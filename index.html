Built for order processing teams

Works with any ERP

SAP, Oracle, D365, NetSuite — Chandlr stages data in Excel. You enter it however you normally do. No API connectors needed.

Smart column detection

Drop in any PO format. Chandlr finds the SKU, price, quantity, description, and UOM columns automatically.

Prefix SKU matching

PO says "1046", ERP says "1046V001"? Chandlr matches them. Case-insensitive, whitespace-tolerant.

Color-coded everything

Green = good. Yellow = check. Red = stop. Every output sheet is instantly readable.

Full audit trail

Reconciliation results, credit notes, re-invoices, and ERP staging — all saved as sheets in the workbook.

9-rule data quality gate

Catches duplicates, missing prices, quantity outliers, and line total mismatches before reconciliation starts.

The intelligence layer

Chandlr is both an Excel add-in and a Microsoft 365 Copilot agent. Use buttons or natural language — your choice.

Copilot declarative agent

Chandlr registers as a Copilot agent in Microsoft 365. Ask it to process orders in plain English — no menus, no training.

Live

Smart extraction

Upload CSV, Excel, or PDF purchase orders. Chandlr auto-detects columns (SKU, price, qty, description, UOM) from any format — no templates, no mapping rules. PDFs are parsed using positional text extraction to reconstruct table structure.

Live

Autonomous pipeline

Each Copilot action chains to the next with a recommended follow-up. The agent walks operators through capture, validate, reconcile, resolve, and ERP staging without switching screens or learning the UI.
